## What This Is

A practical, structured, and anti-hype idea engine for discovering useful AI-powered opportunities. Each idea includes:

- **Target users** — who needs it
- **Pain points** — what problem it solves
- **AI workflow** — what AI does in the solution
- **Business models** — how it could be monetized
- **Difficulty & time to build** — honest effort estimates
- **Validation steps** — how to test before building
- **Risk level** — compliance, market, and technical risks
- **Scoring** — 8-dimension score across pain, market, buildability, and more

## What This Is Not

- Not financial advice or income promises
- Not a "make $10k/month with AI" list
- Not a prompt dump
- Not guaranteed business ideas — every entry is a hypothesis to validate

## Contributing

See [CONTRIBUTING.md](CONTRIBUTING.md) for how to suggest a new idea, report a broken link, or flag hype content.

The fastest way to contribute: [open a new idea suggestion →](https://github.com/msdw/awesome-ai-idea-engine/issues/new?template=suggest-idea.yml)

All submissions go through a human review. Automated discovery creates weekly PRs that a maintainer reviews before merging.

---

## Automated Updates

Every Monday at 07:00 UTC, a GitHub Actions workflow discovers new AI idea candidates from public sources and opens a pull request for human review. Only validated ideas get merged.
